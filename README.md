## 🏛️ About me

**Senior Solutions Architect** with over **11 years of experience** designing cloud platforms, AI infrastructure and MLOps ecosystems in highly regulated environments — **HMRC** (United Kingdom), **BACEN** and **SUSEP** (Brazil). **Four greenfield platforms** delivered across four companies.

I work at the intersection of **Solutions Architecture**, **Enterprise Architecture**, **Platform Engineering**, **Site Reliability Engineering (SRE)** and **Machine Learning Operations (MLOps)**, with deep expertise across **AWS**, **Microsoft Azure** and **Databricks**. From serverless topology design to data governance and FinOps, I own the architectural decisions that shape platforms for years.

### What I do

- 🏛️ **Solutions Architecture** — greenfield cloud platforms designed from the ground up, owning the trade-off decisions that shape MLOps, data and API ecosystems for years.
- ⚙️ **Platform Engineering & SRE** — reproducible, auditable, cost-optimised infrastructure on Terraform, Docker, Kubernetes, AWS Lambda, Amazon ECR, Azure Databricks and Amazon SageMaker.
- 🔁 **CI/CD & DevOps** — Continuous Integration and Continuous Deployment pipelines on GitHub Actions, Azure DevOps, Jenkins and Bitbucket, with PyTest, SonarQube and Snyk.
- 🌐 **API Design & Serverless** — OpenAPI 3.0 contracts and serverless microservices on AWS Lambda, API Gateway, FastAPI and Django, applying Clean Architecture and SOLID.
- 🤖 **MLOps Engineering** — machine learning models operationalised at scale through MLFlow, Unity Catalog, Databricks Model Serving, Amazon SageMaker and Databricks Endpoint API.
- 💸 **FinOps & Data Governance** — cluster policies, autoscaling, Spot instances, lineage through Unity Catalog and medallion architecture (Bronze/Silver/Gold).

### Track record

🇧🇷 **Zurich Santander Insurance** — Built an MLOps platform from scratch on Azure and Databricks for an insurer regulated by BACEN and SUSEP, achieving a **60% reduction in model deployment time-to-market**.

🌎 **AB InBev Global Martech** — Led the architecture of a global data platform serving **23 markets**, consumed by VPs of Marketing for multi-billion-dollar media allocation decisions, with a **45% infrastructure cost reduction**.
